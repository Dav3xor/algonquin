Algonquin Chat:

Algonquin is a chat communication system with wiki-like features, allowing interconnection of user supplied content. It's designed to be user focused, in that it doesn't modify uploaded content, and is not supplied as a service, you need to run your own server/instance of the software.

A script for installing the software to Linode is provided, as is a script for converting a slack backup, if you wish to migrate from there.
